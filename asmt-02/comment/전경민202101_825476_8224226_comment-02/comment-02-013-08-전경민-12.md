# 과제-02 쟁점과 딜레마 분석에 대한 코멘트

- 대상과제: `과제-02 쟁점과 딜레마 분석`
- 코멘트를 제공하는 학생: `013-08 전경민` 
- 코멘트를 받는 학생: `013-12 임서현` 

---

## 코멘트

제출된 쟁점과 딜레마 분석 과제는 넛지 정책이 개인의 자율성과 권력 행위 사이에 어떤 딜레마를 발생시키는지 잘 보여주고 이에 대해 잘 분석한 것 같다. 또, 짧고 간결하게 핵심만 잘 설명하고 있어 읽기에도 큰 부담이 없으며 마지막 '나의 문제 의식' 부분에 두 주장을 어느 정도 절충하되 자신의 의견을 잘 이야기하고 있고 앞으로 어떤 식으로 탐색하고 글을 쓸 것인지 잘 설명하고 있는 것 같다. 다만, 학자의 '상반된 입장' 부분에서 Michael Foucault(1979)의 주장이 자칫 Richard H.,Cass(2008)의 주장과 상관없는 이야기로 읽힐 수 있을 것 같다는 우려가 든다. 경제학적 지식이 조금 부족하더라도 두 학자가 어떤 상반된 주장을 하는지, 어느 부분에서 두 학자의 주장이 어떤 부분에서 충돌하는지 한 번 더 풀어서 설명하거나 부연 설명을 더하여 주면 독자들이 이해하기에 더욱 편할 것 같다. 그리고 두 문헌을 읽어보지는 않아 정확히는 알지 못 하지만 두 문헌이 넛지 정책에 대한 하나의 쟁점에서 대립되는 주장을 하는 것이 아니라, Michael Foucault(1979)의 문헌은 Richard H.,Cass(2008)의 주장을 반박하기 위한 근거에 가까운 것 같다. 같은 쟁점에서 충돌하는 문헌을 다시 찾는다면 더 좋은 글이 될 것 같다.

## 체크 포인트

### **기초 구조 점검**
- [ ] 핵심 쟁점(Core Issue)이 모호하거나 지나치게 넓어서, 문제 설정이 분명하지 않다.
- [ ] 쟁점을 둘러싼 실제 학술 논쟁이 구체적인 문헌과 함께 잘 제시되어 있다.
- [ ] 딜레마의 구조가 두 주장 간의 긴장 또는 선택의 문제로 명확하게 드러나지 않는다.
- [ ] 학자들의 입장이 핵심 쟁점과 딜레마에 밀접하게 연결되어 있다.
- [ ] 관련 학자들의 입장 정리가 단순 나열에 그치고 있으며, 논쟁적 구조(찬반, 대비 등)가 드러나지 않는다.

### **작성자의 문제의식 및 초기 주장**
- [x] 작성자의 문제의식이 쟁점과 딜레마를 반영하여 논리적으로 연결되어 있다.
- [ ] 작성자의 초기 주장이 제시되어 있으나, 그것이 본문에서 제시된 학술 논쟁과 어떤 방식으로 연결되는지가 불분명하다.
- [x] 초기 주장이 학자들의 논의 사이에서 스스로의 입장을 잘 구성하고 있다.

### **개선 방향 제안**
- [ ] 핵심 쟁점을 좁고 구체적으로 설정하여, 실제 논증문에서 다룰 문제를 명확히 할 필요가 있다.
- [ ] 학자 간의 대비되는 입장을 중심으로 논쟁적 구조를 선명히 정리하는 것이 좋다.
- [x] 작성자의 입장을 뒷받침할 수 있는 논문·문헌을 구체적으로 더 보강할 수 있다.
- [ ] 서론이 될 수 있는 문제 배경 서술을 간결하게 재구성해보는 것도 도움이 될 것이다.
